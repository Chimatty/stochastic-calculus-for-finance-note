§1: 無裁定価格評価モデル

無裁定価格評価モデル

一期間二項モデル

一期間のみを考えた二項モデルを定義する。

定義

期間の定義
  • 時刻0: 期間の始まり
  • 時刻1: 期間の終わり

とする。

コインの定義

一つのコインがあり、コインの表が出る確率がppp、裏が出る確率がq=1−pq = 1 - pq=1−pであるとする。

株式マーケットの定義

SSSという株式のマーケットがあるとする。

株式SSSの価格は

  • 時刻0ではS0S_0S​0​​
  • 時刻1では、コイン投げの結果に依存し、表が出ればS1(H)S_1(H)S​1​​(H)、裏が出ればS1(T)S_1(T)S​1​​(T)

とする。

便宜のため、

  • 上昇率 u:=S1(H)/S0u := S_1(H)/S_0u:=S​1​​(H)/S​0​​
  • 下落率 d:=S1(T)/S0d := S_1(T)/S_0d:=S​1​​(T)/S​0​​

とする。

マネーマーケットの定義

株式SSSとは別に、コイン投げに依存しないで価格が決まるマーケットをマネーマーケットというものを考える。一般には国債のようなデフォルトリスクがないものをイメージすれば良い。マネーマーケットは金利rrrが存在し、時刻0におけるマネーマーケットの1ドルは、時刻1において1+r1+r1+rドルになるとする。

無裁定条件の仮定
  • 無裁定条件 0<d<1+r<u0 < d < 1 + r < u0<d<1+r<u

を仮定する。(無裁定条件について詳しくは本を参照)
